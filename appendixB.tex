\chapter{Mention Count Source List}
\label{appendix:mention-count-source-list}

This appendix lists the sources used to construct the consolidated mention-count ranking described in Section~\ref{sec:mention-count-ranking}. Sources fall into three groups: community-curated robotics software lists, academic survey or comparison papers, and auxiliary LLM-assisted discovery lists. The LLM-assisted lists were used only as supplementary discovery aids, as discussed in Section~\ref{subsec:llm-assisted-sources}.
